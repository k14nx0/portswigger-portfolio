# 🛡️ Web Application Penetration Testing Showcase
An interactive repository containing comprehensive, hands-on vulnerability research and write-ups from the PortSwigger Web Security Academy. This portfolio covers practical lab exploitation methodologies mapping directly to critical entries in the OWASP Top 10 framework.

---

### 🚀 Active Security Research Modules

Click into any individual directory module folder below to explore full step-by-step attack chains, vulnerability analyses, and embedded technical screenshots:

* **[🔹 Lab 01: Broken Access Control (Insecure Parameter Manipulation)](./lab01-broken-access-control)**  
  Exploits unvalidated client-side state tracking cookies to manually forge authorization tokens, bypass perimeter authentication gates, and escalate user privileges directly into hidden administrative dashboards.

* **[🔹 Lab 02: Authentication Bypass (Automated Username Enumeration)](./lab02-username-enumeration-different-responses)**  
  Leverages subtle variations in validation error-handling logic layout sizes to extract active database profile metrics using custom dictionary automation arrays via Burp Suite Intruder.

* **[🔹 Lab 03: Server-Side Request Forgery (SSRF Against Local Infrastructure)](./lab03-basic-ssrf-local-server)**  
  Coerces trusted external API parameters into launching internal network loopback request pathways natively on behalf of the attacker, effectively defeating access-control lists behind the firewall border perimeter.
